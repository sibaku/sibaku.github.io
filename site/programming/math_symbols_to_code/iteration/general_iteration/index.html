General Iteration rule

As all of the previous symbols follow the same rules as the sum and product, we will write down a general rule (sometimes people might even define other symbols that work the same way!) and then tell you how these new symbols work.

For each iteration symbol we define an initial value vvv and an operation op⁡\operatorname{op}op that combines that value with a term in the iteration to produce the new value. Here is a list of the so far discussed symbols:

  • ∑\sum∑: v=0v=0v=0, op⁡=+\operatorname{op} = +op=+
  • ∏\prod∏: v=1v=1v=1, op⁡=∗\operatorname{op} = *op=∗
  • ⋃\bigcup⋃: v=∅v= \emptysetv=∅ (empty set), op⁡=∪\operatorname{op} = \cupop=∪
  • ⋂\bigcap⋂: v=Av= Av=A (set of all elements), op⁡=∩\operatorname{op} = \capop=∩
    • Note: For this operation you would probably just start with the first element and handle no elements as a special case

Then the general code will look like this:

let v = ...

for(let i = 0; i <= n; i++){
    v = op(v,f(i));
}

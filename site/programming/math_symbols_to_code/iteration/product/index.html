Product ∏\prod∏

The product symbol ∏\prod∏ specifies a product of multiple terms. The symbol is a capital greek letter "Pi", so Pi for product.

Basic form ∏i=0nfi\prod\limits_{i=0}^n f_ii=0∏n​fi​ or ∏i=0nfi\prod_{i=0}^n f_i∏i=0n​fi​

Parts:

  • i=0i=0i=0: Name of the index with initial value
  • nnn: Last value of the named index. ATTENTION! The last index is inclusive, so iii will take on that value and not stop one before that
  • fif_ifi​: The iii-th product term. This could be any expression using (or not using!) iii

Meaning: Multiply each term fif_ifi​, starting at i=0i=0i=0 and ending at i=ni=ni=n. If the upper stop value is lower than the initial value, then the sum is 111, e.g. ∏i=10=1\prod\limits_{i=1}^0 = 1i=1∏0​=1.

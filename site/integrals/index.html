Introduction

Integrals are found in many different applications.

A very common one is finding the distance traveled while moving with a varying velocity, which can also be described as finding the length of some path that you moved on. You might unknowingly encounter this in many games. If a camera in a cutscene or an NPC patrolling an area need to move at a certain speed along a designer specified path, you need to calculate the length of it first.

You can find the area of complicated shapes with integrals, which uses the classic notion of area under a graph.

Probability theory uses integrals for defining various kinds of events and combinations of events.

In computer graphics we can define the interaction of light and surfaces with integrals.

Many applications go farther than the basic notation and examples we will see in the following sections, but are fundamentally the same.

We won't do any overly formal or perfect proofs in this document, there are other documents for that and I highly encourage you to check those out if you really want to understand everything on a fundamental level. Instead we will try to find some kind of understanding of the notation and words commonly used, hopefully striking a balance between accuracy and understandability for someone not fully into math formalism. Feel free to tell me if there is something actually really wrong though, I would like to not convey misinformation!

Area under a curve

The first time you will encounter integrals is probably when trying to find the area under a graph defined by some function f⁡\operatorname{f}f. We will also use this approach, as it is pretty intuitive.

Basic idea

Let's start with a very simple function: f⁡(x)=1\operatorname{f}(x) = 1f(x)=1, which is a horizontal line, that looks as follows:

Now, how would we calculate the area under that graph between 000 and 222? That area looks like a rectangle, so why not use that. The area of a rectangle is just width times height. Height is given by f⁡\operatorname{f}f, in this case just 111. The width is the xxx-range of the regions, so 2−0=22-0=22−0=2, giving us an area of 2∗1=22*1=22∗1=2.

Now let's look at a slightly more complicated function f⁡(x)=2x\operatorname{f}(x) = 2xf(x)=2x in the same region.

We can of course again use a rectangle, but this won't give us the exact area in this case due to the non horizontal line. We can still try to approximate the area. But how to choose the height? Well, there are infinite choices. Let's stick with two simple choices: Use the function value at the left or right side, so f⁡(0)\operatorname{f}(0)f(0) and f⁡(2)\operatorname{f}(2)f(2).

Obviously, the left rectangle has an area of 000, since 2∗0=02*0 = 02∗0=0, so it shows up as just a line. The right rectangle then has an area of 2∗2∗2=4∗2=82*2 * 2= 4 * 2 = 82∗2∗2=4∗2=8. Now we can actually compute the area ourselves, since the shape is just right-angled triangle! For that, the formula is just half the height times the base, so 2∗2∗22=4\frac{2*2*2}{2} = 422∗2∗2​=4. This of course makes sense, as it is exactly half of the right rectangle. but we can also see, that the solutions lies between our two approximations.

How to do better? Well, how about we keep using rectangles, since they are really easy to calculate, but we use more of them. Let's start by using two rectangles, one for each half of the region. Defining what the integral is

In the last sections, we have seen how we can approach computing the area under a curve by just drawing a bunch of easy to calculate rectangles along the function.

In this section we will write it down nicely and cover the notation you might come across when you look up integrals, which might seem intimidating.

We start by defining an interval in which we want to find the area of the function, denoted by [a,b][a,b][a,b]. aaa is the lower starting xxx-value, bbb is the upper ending xxx-value. Since we are planning to make a bunch of rectangles, let's use a name for the number of those rectangles: nnn.

Since we try to keep it easy, each rectangle will have the same width. The length of the total xxx interval is b−ab-ab−a. Thus, the width of one rectangle is just b−an\frac{b-a}{n}nb−a​. We should also give this value a name:

Δx=b−an\Delta x = \frac{b-a}{n} Δx=nb−a​

This might be a first stumbling block, but don't worry! That symbol Δ\DeltaΔ is a Greek letter and you pronounce it "delta", so Δx\Delta xΔx is pronounced "delta x". This Δ\DeltaΔ symbol is commonly used in math and physics to denote some kind of difference (Delta, Difference). Here we also have a difference, though scaled a bit.

We will also need xxx-values, where we evaluate the function at to get the height of our rectangles. As we only use xxx-values at the beginning or the end of our rectangles, we should just find some expression for those. The easiest ones are the first and last one, those are just x0=ax_0 = ax0​=a (first) and xn=bx_n = bxn​=b (last). The subscript will become clearer in the next step. Now, what is the second value? It should be one rectangle width away from the first. We already defined the rectangle width as Δx\Delta xΔx!. So the second value is x1=x0+Δxx_1 = x_0 + \Delta xx1​=x0​+Δx. The third one then is one width further than the second one. x2=x1+Δxx_2 = x_1 + \Delta xx2​=x1​+Δx. We can also just put in the definition of x1x_1x1​ into that last one: x2=x1+Δx=x0+Δx+Δx=x0+2Δxx_2 = x_1 + \Delta x = x_0 + \Delta x + \Delta x = x_0 + 2\Delta xx2​=x1​+Δx=x0​+Δx+Δx=x0​+2Δx.

You can probably guess the pattern from that. x3x_3x3​ will be x0+3Δxx_0 + 3\Delta xx0​+3Δx and so on. If we use that subscript as a nice index for which point we are talking about, then we get this very simple formula:

xi=x0+iΔxx_i = x_0 + i\Delta x xi​=x0​+iΔx

This is consistent with the start of this. If we plug in i=0i=0i=0, we get x0x_0x0​. If we plug in i=ni=ni=n (and remember that x0=ax_0 = ax0​=a), we get x0+nΔx=a+nb−an=a+b−a=b=xnx_0 + n\Delta x = a + n\frac{b-a}{n} = a + b - a = b = x_nx0​+nΔx=a+nnb−a​=a+b−a=b=xn​.

We can now define the area of one of our rectangles. Let's start with the first one again. We have to decide, whether to use the left or right rectangle. If we use the left, then we evaluate the function f⁡\operatorname{f}f, that we want to integrate, at the left point. So the area will be f⁡(x0)∗Δx\operatorname{f}(x_0) * \Delta xf(x0​)∗Δx. The second rectangle will then use the second xxx-value, and its area is f⁡(x1)∗Δx\operatorname{f}(x_1) * \Delta xf(x1​)∗Δx. This will go on until the nnn-th rectangle. If you keep in mind, that we started counting at 000 for our xxx subscripts, the last rectangle will be f⁡(xn−1)∗Δx\operatorname{f}(x_{n-1}) * \Delta xf(xn−1​)∗Δx. In the following, let's skip the ∗*∗ symbol, as it clutters up the notation, so we write f⁡(x0)Δx\operatorname{f}(x_0)\Delta xf(x0​)Δx instead of f⁡(x0)∗Δx\operatorname{f}(x_0) * \Delta xf(x0​)∗Δx. Let us now add up all our areas to find the total area AAA!

A=f⁡(x0)Δx+f⁡(x1)Δx+f⁡(x2)Δx+⋯+f⁡(xn−1)ΔxA = \operatorname{f}(x_0)\Delta x + \operatorname{f}(x_1)\Delta x + \operatorname{f}(x_2)\Delta x + \dots + \operatorname{f}(x_{n-1})\Delta x A=f(x0​)Δx+f(x1​)Δx+f(x2​)Δx+⋯+f(xn−1​)Δx

As you can see, all the terms in the sum look basically the same. Each term looks like f⁡(xi)Δx\operatorname{f}(x_i)\Delta xf(xi​)Δx, where iii is some index. We basically just write down a list of values for iii and then add the corresponding term to the total. The values that iii takes are 1,2,3,…,n−11,2,3,\dots,n-11,2,3,…,n−1. With this we introduce the next math notation you will come across very often. The summation symbol ∑\sum∑ (Sigma). With that symbol, the above sum can be written compactly as:

A=∑i=0n−1f⁡(xi)ΔxA = \sum_{i=0}^{n-1}\operatorname{f}(x_i)\Delta x A=i=0∑n−1​f(xi​)Δx

Below the ∑\sum∑, we write the name of the index variable, here iii, and what its start index is, here 000. On top of the ∑\sum∑ we write what the final value of iii will be, here n−1n-1n−1. This then just means: Write down all integer values of iii between the starting and endvalue (inclusive), plug that value into the formula right of ∑\sum∑ and add all the different values that are produced that way. Which is exactly what we have written down before. You can remember the connection of the symbol and sums as: Sigma, Sum.

We can also see how nice this notation is, when we now consider the right rectangles. They only differ, in that we start with not the first xxx-value, but the second one, namely x1x_1x1​. We still have the same number of rectangles. As a consequence, the last rectangle will use the last xxx-value xnx_nxn​, whereas the left rectangles used the second to last. As an exercise, you can verify the following, by writing it down for some nnn, for example n=4n=4n=4. The total area for the right rectangles can be written as:

A=∑i=1nf⁡(xi)ΔxA = \sum_{i=1}^{n}\operatorname{f}(x_i)\Delta x A=i=1∑n​f(xi​)Δx

Looks basically the same, just shifted by one!

Only one last step is missing to arrive at what the integral will be!

As we have seen in the last section, increasing nnn will make our two sums become closer to the correct solution. What we want is for both of them to agree on the result. For that, we have to make it so small, that it won't matter anymore, if we choose the left or right side as the height. The higher the number of rectangles nnn, the smaller the width Δx\Delta xΔx. The idea is now to not just use a finite number of rectangles, but infinite ones. This is probably a bit hard to think about. From a practical point of view, think about how the Δx\Delta xΔx will become smaller and smaller for very large numbers of nnn. Now if you had a really good microscope to look at the graph, at some point you probably couldn't differentiate between different rectangles anymore, as they were too small. If your function f⁡\operatorname{f}f is not some incredibly weird one, it shouldn't change that much in value anymore inside of that super small rectangular region, at least your microscope wouldn't be able to see anything. At that point, you wouldn't care about left or right rectangle, you would measure the same value for both.

In math, we have a tool to describe this process of "make a value go towards infinity and see what happens". It is called the "limit". We write the limit of an expresion expr\text{expr}expr as nnn goes to infinity as lim⁡n→∞expr\lim_{n\rightarrow \infty} \text{expr}limn→∞​expr. Basically we check, if the expression will result in a nice value when considering an infinite value. Writing this down for our area AAA (right rectangles), we have:

A=lim⁡n→∞(∑i=1nf⁡(xi)Δx)A = \lim_{n\rightarrow \infty}(\sum_{i=1}^{n}\operatorname{f}(x_i)\Delta x) A=n→∞lim​(i=1∑n​f(xi​)Δx)

If that expression does make sense, we introduce a nice new formulation for that whole thing:

∫abf⁡(x)dx=lim⁡n→∞(∑i=1nf⁡(xi)Δx)\int_a^b\operatorname{f}(x)dx = \lim_{n\rightarrow \infty}(\sum_{i=1}^{n}\operatorname{f}(x_i)\Delta x) ∫ab​f(x)dx=n→∞lim​(i=1∑n​f(xi​)Δx)

The Δx\Delta xΔx basically becomes dxdxdx, the symbol students are always reminded about not to forget. The ddd part her signals the process of going from the discrete fixed number of rectangles to infinite ones, each with a width that isn't really a number anymore. We often call this thing a "differential" value. Still, the dxdxdx holds this concept of the width that we used before, while the f⁡(x)\operatorname{f}(x)f(x) part holds the concept of height, so we can't forget about either of them when writing down an integral.

Similarly to the analogy with the microscope, you can think about subdividing that ∑\sum∑ symbol in smaller and smaller regions. At the beginning, it's just four lines. But if you keep refining it with more and more lines and smooth them out a bit, your ∑\sum∑ symbol might look something like ∫\int∫. It is basically just a smooth S, just as in Sum and Sigma. You can think about the ∫\int∫ as the symbol that incapsulates the whole processing of splitting up the area in incredibly small parts and adding them.

Now, with those last few paragraphs you might be asking: What? How would we even compute an integral with infinities flying around? And you wouldn't be wrong. You can actually find results with that definition, but it is tedious and probably won't be getting you anywhere for anything non-trivial. These kinds of constructions are very important for creating certain formulations and proving properties though.

Which is why in the next section we will go over the "common" way to compute an integral.

Computing integrals

This section will cover actually computing integrals. First we will just give a brief overview of numerical methods, no details, just some keywords. Then we will go to antiderivatives and how they work.

Numerical computation

From the previous sections, we basically already have one method of computing an integral, albeit not exactly in most cases. If we take the right hand side of the integral and remove the limit, but exchange the equality with an approximate equality, we have out first method:

∫abf⁡(x)dx=lim⁡n→∞(∑i=1nf⁡(xi)Δx)≈∑i=1nf⁡(xi)Δx\begin{align*} \int_a^b\operatorname{f}(x)dx &= \lim_{n\rightarrow \infty}(\sum_{i=1}^{n}\operatorname{f}(x_i)\Delta x) \\ &\approx \sum_{i=1}^{n}\operatorname{f}(x_i)\Delta x \end{align*} ∫ab​f(x)dx​=n→∞lim​(i=1∑n​f(xi​)Δx)≈i=1∑n​f(xi​)Δx​

We have to choose a number of segments and then just calculate the summation terms, as we did in the beginning. This is annoying to do by hand, so in most cases we will use a computer, which can calculate far higher number of segments in very short amounts of time. It is also straightforward, just requiring basic arithmetic and a loop.

Still, we might want to keep the number of segments as low as possible, for example if the function is very costly to evaluate. Other schemes exist to find better approximations, such as the trapezoidal or Simpson's rule.

A very different approach is to use stochastics. As mentioned in the introduction, integrals play a huge part in probability theory. The definition of the expected value of a function actually allows us to compute an approximation of an integral with random numbers!

Antiderivatives

While not always possible, antiderivatives provide a very powerful way to compute integrals. The following will cover the basic idea on the second fundamental theorem of calculus, which allows us to directly write down integrals, that we know are derivatives of some other function.

For now we will not cover other techniques, such as substitution.

The fundamental theorem of calculus

The fundamental theorem of calculus relates antiderivatives with integrals. As stated before, this is not a math textbook, so please consult one for the rigorous definitions of this all, such as the correct intervals which we technically need to be careful about.

First of, what is an antiderivative? Usually, it is just written with the upper case letter of the function to be considered.

If the following statement is true, then F⁡\operatorname{F}F is an antiderivative for f⁡\operatorname{f}f.

F⁡(t)=∫atf⁡(x)dx \operatorname{F}(t) = \int_a^t \operatorname{f}(x)dx F(t)=∫at​f(x)dx

And antiderivative means:

F⁡′(x)=f⁡(x) \operatorname{F}'(x) = \operatorname{f}(x) F′(x)=f(x)

Now one cause of confusion might be the different variable name in the first statement, where we used F⁡(t)\operatorname{F}(t)F(t). This is basically just to distinguish different variables. The xxx and dxdxdx are basically placeholder names and stand for the different terms in our integral sum (just that the sum isn't discrete anymore). We could call them by any way we want. The ttt in the first statement is the parameter of F⁡\operatorname{F}F. It effects the upper bound of the integral. That basically just means: Only calculate the sum, until your xxx value reaches ttt. In the second statement, we use the same variable xxx since we actually mean the same position. In words: The rate of change at xxx is just the value of f⁡\operatorname{f}f at xxx.

We want to see why that is true. But we can already think about why it intuitively makes sense:

The derivative measures the rate of change of a function. F⁡(x)\operatorname{F}(x)F(x) describes the area of the curve up until xxx. Now, what exactly causes F⁡(x)\operatorname{F}(x)F(x) to change? The function f⁡\operatorname{f}f of course. Similarly to when we walk. Let's say you have walked 1m1m1m up until time ttt. Now you walk with a speed of 2ms2\frac{m}{s}2sm​. If someone asked you what the rate of change was between time ttt and t+1st+1st+1s. You would just check how far you have gotten in that time. After a second of movement, you have moved 2m2m2m. And the rate of change was thus 2ms2\frac{m}{s}2sm​, your speed.

Continuing from our discussion from before, we can do a very non-rigorous "proof" to check how this works out. This should be roughly correct, but it is not in the details, please be aware of that.

So for simplicity, we will just use ttt values located at one of our segment endpoints xix_ixi​ (inbetween is just a constant rectangle anyways). We might not have an exact expression for F⁡\operatorname{F}F, but we have approximated the integral in the last section!

We will evaluate F⁡\operatorname{F}F at some segment point xjx_jxj​, where jjj is just some value between 111 and nnn.

F⁡(xj)=∫axjf⁡(x)dx≈∑i=0j−1f⁡(xi)Δx\begin{align*} \operatorname{F}(x_j) &= \int_a^{x_j}\operatorname{f}(x)dx \\ &\approx \sum_{i=0}^{j-1}\operatorname{f}(x_i)\Delta x \end{align*} F(xj​)​=∫axj​​f(x)dx≈i=0∑j−1​f(xi​)Δx​

So to get the integral up to the value xjx_jxj​ we just don't use all the terms until nnn, but stop at j−1j-1j−1 instead for the left rectangles. The −1-1−1 is due to the same reason as before: We use the value left of our segment as height.

Now we try to differentiate F⁡\operatorname{F}F, which might sound scarier than it is.

You may recall the secant approximation for the derivative, called the difference quotient:

F⁡′(xj)≈F⁡(xj+Δx)−F⁡(xj)Δx \operatorname{F}'(x_j) \approx \frac{\operatorname{F}(x_j + \Delta x) - \operatorname{F}(x_j)}{\Delta x} F′(xj​)≈ΔxF(xj​+Δx)−F(xj​)​

We look a bit ahead of the point xjx_jxj​, calculate the change from our current position xjx_jxj​ and divide by the distance traveled between the two points, to obtain the slope. As a further simplification, we use the same distance that we use in our rectangles Δx\Delta xΔx.

Now let's plug in our expression for F⁡\operatorname{F}F!

F⁡′(xj)≈F⁡(xj+Δx)−F⁡(xj)Δx≈∑i=0jf⁡(xi)Δx−∑i=1j−1f⁡(xi)ΔxΔx\begin{align*} \operatorname{F}'(x_j) &\approx \frac{\operatorname{F}(x_j + \Delta x) - \operatorname{F}(x_j)}{\Delta x} \\ &\approx \frac{\sum_{i=0}^{j}\operatorname{f}(x_i)\Delta x - \sum_{i=1}^{j-1}\operatorname{f}(x_i)\Delta x}{\Delta x} \end{align*} F′(xj​)​≈ΔxF(xj​+Δx)−F(xj​)​≈Δx∑i=0j​f(xi​)Δx−∑i=1j−1​f(xi​)Δx​​

As we used Δx\Delta xΔx as our distance, our approximation for F⁡(xj+Δx)\operatorname{F}(x_j + \Delta x)F(xj​+Δx) can just walk one step farther than jjj, as you can see in the sum.

Now when you look at the two sums in the numerator, they are equal aside from the +1+1+1 on the top of the first ∑\sum∑. So aside from the term for jjj, all other terms appear on the left and the right side. And since the right side is subtracted from the left, we are left with:

F⁡′(xj)≈F⁡(xj+Δx)−F⁡(xj)Δx≈∑i=0jf⁡(xi)Δx−∑i=0j−1f⁡(xi)ΔxΔx=f⁡(xj)ΔxΔx=f⁡(xj)\begin{align*} \operatorname{F}'(x_j) &\approx \frac{\operatorname{F}(x_j + \Delta x) - \operatorname{F}(x_j)}{\Delta x} \\ &\approx \frac{\sum_{i=0}^{j}\operatorname{f}(x_i)\Delta x - \sum_{i=0}^{j-1}\operatorname{f}(x_i)\Delta x}{\Delta x} \\ &= \frac{\operatorname{f}(x_{j})\Delta x}{\Delta x} \\ &= \operatorname{f}(x_{j}) \end{align*} F′(xj​)​≈ΔxF(xj​+Δx)−F(xj​)​≈Δx∑i=0j​f(xi​)Δx−∑i=0j−1​f(xi​)Δx​=Δxf(xj​)Δx​=f(xj​)​

And with that we have arrived at what the fundamental theorem of calculus says: The derivative of the antiderivative is the function inside of the integral!

The important consequence is: If we know the derivative of a function, then that function is an antiderivative of the derivative!

Derivatives are generally much easier than antiderivatives. With this property, we can find a lot of base antiderivatives with wich more complex ones can be constructed with other means. Neat!

One last thing to note here is another one of those parts, that you easily get points deducted for in tests!

Let's say you know an antiderivative F⁡(x)\operatorname{F}(x)F(x). Now we add some constant value ccc to it: F⁡(x)+c\operatorname{F}(x) + cF(x)+c. What happens, if we differentiate that?

(F⁡(x)+c)′=F⁡′(x)+c′=f⁡(x)+0=f⁡(x)\begin{align*} (\operatorname{F}(x) + c)' &= \operatorname{F}'(x) + c' \\ &= \operatorname{f}(x) + 0 \\ &= \operatorname{f}(x) \end{align*} (F(x)+c)′​=F′(x)+c′=f(x)+0=f(x)​

As constants vanish under differentiation, there are actually infinitely many antiderivatives, each differing by some constant factor to all the others. Which is why you should write down the +c+c+c, when writing down antiderivatives.

Now how do we solve the problem, that our solution is not unique? Well, we actually know one correct value of F⁡\operatorname{F}F, namely F⁡(a)+c=∫aaf⁡(x)dx=0\operatorname{F}(a) +c = \int_a^a \operatorname{f}(x)dx = 0F(a)+c=∫aa​f(x)dx=0. This means, that the area from aaa to aaa, a zero length interval is 000. We can use that to nail down ccc:

F⁡(a)+c=0c=−F⁡(a)\begin{align*} \operatorname{F}(a) + c &= 0 \\ c &= -\operatorname{F}(a) \end{align*} F(a)+cc​=0=−F(a)​

And we can then compute the integral as

∫abf⁡(x)dx=F⁡(b)−F⁡(a) \int_a^b \operatorname{f}(x)dx = \operatorname{F}(b) - \operatorname{F}(a) ∫ab​f(x)dx=F(b)−F(a)

The second fundamental theorem of calculus, which states, that if F⁡\operatorname{F}F is an antiderivative of f⁡\operatorname{f}f, then we have:

∫abf⁡(x)dx=F⁡(b)−F⁡(a) \int_a^b \operatorname{f}(x)dx = \operatorname{F}(b) - \operatorname{F}(a) ∫ab​f(x)dx=F(b)−F(a)

results in the same expression, but starts from a general antiderivative.

We also use another shorthand notation:

F⁡(b)−F⁡(a)=[F⁡(x)]ab\operatorname{F}(b) - \operatorname{F}(a) = [\operatorname{F}(x)]_a^b F(b)−F(a)=[F(x)]ab​

Examples

Here are a few examples, that you will usually get taught alongside antiderivatives. Basically we calculate some derivatives and from that get some antiderivatives.

One of the first types of functions you will encounter are polynomials, which are luckily easy to differentiate. The derivative of xnx^nxn is just nxn−1nx^{n-1}nxn−1. From this we try to go the other way around. Can we find a polynomial, that works just like this rule and results in xnx^nxn? Well, yes! 1n+1xn+1\frac{1}{n+1}x^{n+1}n+11​xn+1. We can check the rule:

(1n+1xn+1)′=(n+1)1n+1xn=xn\begin{align*} (\frac{1}{n+1}x^{n+1})' &= (n+1)\frac{1}{n+1}x^n \\ &= x^n \end{align*} (n+11​xn+1)′​=(n+1)n+11​xn=xn​

This checks out and so we have:

∫abxndx=[1n+1xn+1]ab \int_a^b x^n dx= [\frac{1}{n+1}x^{n+1}]_a^b ∫ab​xndx=[n+11​xn+1]ab​

Another common example is exe^xex, which is especially nice, since (ex)′=ex(e^x)' = e^x(ex)′=ex. So we also have:

∫abexdx=[ex]ab \int_a^b e^xdx = [e^x]_a^b ∫ab​exdx=[ex]ab​

And as the last example, we have the sine and cosine functions, which derive into each other. We have:

sin⁡′(x)=cos⁡(x)cos⁡′(x)=−sin⁡(x) \sin'(x) = \cos(x)\\ \cos'(x) = -\sin(x) sin′(x)=cos(x)cos′(x)=−sin(x)

This results in the following two integrals:

∫absin⁡(x)dx=[−cos⁡(x)]ab∫abcos⁡(x)dx=[sin⁡(x)]ab \int_a^b \sin(x)dx = [-\cos(x)]_a^b\\ \int_a^b \cos(x)dx = [\sin(x)]_a^b ∫ab​sin(x)dx=[−cos(x)]ab​∫ab​cos(x)dx=[sin(x)]ab​

Geometry

Here we have demos related to geometry.

Inscribed angle

Visualization of the Inscribed angle theorem.

The code can be seen here in the function demoInscribedAngle.

Triangle circles

Visualization of the circles related to a triangle.

The code can be seen here in the function demoTriangleCircles.

Arc tangents

Showcases functionality to compute the various tangents against arcs, which may be circles, open circles as well as ellipses.

The code can be seen here in the function demoArcTangents.

Trigonometry

Shows the usual trigonometric functions in the unit circle.

The code can be seen here in the function demoTrig.

Pythagorean theorem

Visualization of the Pythagorean theorem.

The code can be seen here in the function pythagoras.

Rectangle

Simple visualization of a rectangle, which can be adjusted by the user.

The code can be seen here in the function controllableRectangle.

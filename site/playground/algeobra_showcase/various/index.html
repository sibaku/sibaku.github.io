Various

Here we have demos that don't easily fit into the other categories.

Minkowski difference

Visualization of the Minkowski difference of two polygons. This can be used to check for collisions. If the two shapes overlap, the Minkowski difference will contain the origin.

The code can be seen here in the function demoMinkowskiDifference.

Clipping

Algeobra Showcase

Here you can find a number of showcases and demos to demonstrate my geogebra-like tool Algeobra to create geometric (and general) descriptions of scenes which can then be displayed.

The code for these is linked where they appear. Each showcase contains a very brief description. Generally, each demo is interactive in some way, so try clicking on stuff or fiddle with the controls!

Taylor Series

We will now have a look at how to approximate a function, given that we know its derivatives at a point. Note that we will not cover topics such as the existence of such an approximation or whether it actually converges to the function itself. Such and more details are best sought out in an actual math course. There are quite a few free resources online both in text/textbook as well as video form!

To understand the following, you only need to have some basic knowledge of functions and how to calculate derivatives.

For now consider just some normal function f(x)f(x)f(x) that we can keep differentiating. While this function is nice, for various reasons we might try to find a different expression to represent it. Maybe for some theoretical reason or maybe just to compute it in a different way.

One important class of functions are polynomials, the sums of different powers of xxx with a coefficient for each entry. It looks like this:

p⁡(x)=a0+a1x+a2x2+a3x3+…\operatorname{p}(x) = a_0 + a_1 x + a_2 x^2 + a_3 x^3 + \dots p(x)=a0​+a1​x+a2​x2+a3​x3+…

As a quick reminder, the derivative of the term axkax^kaxk is akxk−1akx^{k-1}akxk−1, e.g. (2x4)′=8x3(2x^4)' = 8x^3(2x4)′=8x3. The derivative of a sum is just the sum of the derivatives of each term. This allows us to easily derive polynomials.

You probably remember from math classes, that the value of the first derivative at a point will give you the slope of the tangent touching the function at that point. That is, a linear function that, if you look very closely at the point, will roughly coincide with the original function in that small region. Higher derivatives are pretty similar, just that they don't correspond to a line, but represent the curving around of the function. The higher the derivative, the more precise the type of wiggling it describes.

As an example, look at a quadratic polynomial p⁡(x)=a0+a1x+a2x2\operatorname{p}(x) = a_0 + a_1 x + a_2x^2p(x)=a0​+a1​x+a2​x2. When we take the second derivative we get the following:

p⁡(x)=a0+a1x+a2x2p⁡(x)′=a1+2a2xp⁡(x)′′=2a2 \begin{align*} \operatorname{p}(x) &= a_0 + a_1 x + a_2x^2 \\ \operatorname{p}(x)' &= a_1 + 2a_2x \\ \operatorname{p}(x)'' &= 2a_2 \end{align*} p(x)p(x)′p(x)′′​=a0​+a1​x+a2​x2=a1​+2a2​x=2a2​​

We got (with a factor of 222) the coefficient of the quadratic term, which regulates how the parabola of a quadratic function looks like.

The basic idea is, that we want our function f⁡\operatorname{f}f to agree with the value of p⁡\operatorname{p}p at a certain point. But we also want all of their derivatives at that point to agree! That way they "wiggle" the same way and thus should probably look the same in a region around that point.

How can we encode that in math?

First, let's choose an easy point to evaluate: x=0x=0x=0.

Now f⁡\operatorname{f}f and p⁡\operatorname{p}p should have the same value at x=0x=0x=0:

f⁡(0)=p⁡(0)=a0+a10+a202+a303+…=a0 \begin{align*} \operatorname{f}(0) &= \operatorname{p}(0) \\ &= a_0 + a_1 0 + a_2 0^2 + a_3 0^3 + \dots \\ &= a_0 \end{align*} f(0)​=p(0)=a0​+a1​0+a2​02+a3​03+…=a0​​

All the terms with xxx vanish at 000! This leaves only the first coefficient. And with that, we know that we should choose a0=f⁡(0)a_0 = \operatorname{f}(0)a0​=f(0).

Let's look at a function, say exe^xex. The value at 000 is 111, so we have a horizontal line at y=1y=1y=1, which is the polynomial with degree zero: p⁡(x)=1\operatorname{p}(x) = 1p(x)=1:

Let's go further. Let the first derivatives agree as well. First let's compute the first derivative of p⁡\operatorname{p}p.

p⁡′(x)=(a0+a1x+a2x2+a3x3+a4x4… )′=a1+2a2x+3a3x2+4a4x3… \begin{align*} \operatorname{p}'(x) &= (a_0 + a_1 x + a_2 x^2 + a_3 x^3 + a_4x^4 \dots)'\\ &= a_1 + 2a_2 x + 3a_3 x^2 + 4a_4x^3 \dots \end{align*} p′(x)​=(a0​+a1​x+a2​x2+a3​x3+a4​x4…)′=a1​+2a2​x+3a3​x2+4a4​x3…​

Then add our condition:

f⁡′(0)=p⁡′(0)=a1+2a20+3a302+4a403+…=a1 \begin{align*} \operatorname{f}'(0) &= \operatorname{p}'(0) \\ &=a_1 + 2a_2 0 + 3a_3 0^2 + 4a_40^3 + \dots \\ &= a_1 \end{align*} f′(0)​=p′(0)=a1​+2a2​0+3a3​02+4a4​03+…=a1​​

As before, we are only left with one non-zero term: a1=f⁡′(0)a_1 = \operatorname{f}'(0)a1​=f′(0).

We can look at the previous example and note the nice property, that (ex)′=ex(e^x)' = e^x(ex)′=ex. So we have a1=e0=1a_1 = e^0 = 1a1​=e0=1. And with that a polynomial of degree 111 with p⁡(x)=1+x\operatorname{p}(x) = 1 + xp(x)=1+x

We will do one last step before the general version. Do a second derivative, that is derive the first derivative again.

p⁡′′(x)=(p⁡′(x))′=(a1+2a2x+3a3x2+4a4x3… )′=2a2+6a3x+12a4x2… \begin{align*} \operatorname{p}''(x) &= ( \operatorname{p}'(x))'\\ &= (a_1 + 2a_2 x + 3a_3 x^2 + 4a_4x^3 \dots)'\\ &= 2a_2 + 6a_3 x + 12a_4x^2 \dots \end{align*} p′′(x)​=(p′(x))′=(a1​+2a2​x+3a3​x2+4a4​x3…)′=2a2​+6a3​x+12a4​x2…​

The next step is basically the same as before, but has a slight difference:

f⁡′′(0)=p⁡′′(0)=2a2+6a30+12a402…=2a2f⁡′′(0)2=a2 \begin{align*} \operatorname{f}''(0) &= \operatorname{p}''(0) \\ &=2a_2 + 6a_3 0 + 12a_4 0^2 \dots \\ &= 2a_2 \\ \frac{\operatorname{f}''(0)}{2} &= a_2 \end{align*} f′′(0)2f′′(0)​​=p′′(0)=2a2​+6a3​0+12a4​02…=2a2​=a2​​

This time, we have a factor of 12\frac{1}{2}21​!

Continuing with exe^xex and that it stays the same under differentiation, we have a2=e02=12a_2 = \frac{e^0}{2} = \frac{1}{2}a2​=2e0​=21​. This gives us the quadratic polynomial p⁡(x)=1+x+x22\operatorname{p}(x) = 1 + x + \frac{x^2}{2}p(x)=1+x+2x2​.

As you can see, it starts to look more like the actual function!

If we do the same thing for the third derivative, you can most likely see that the next factor will be 16\frac{1}{6}61​, since 666 is the factor added from the repeated differentiation.

When we now think about some generic term in the sum, let's say it is the nnn-th one: anxna_nx^nan​xn. When will we get to the same point as with the previous examples, where only the coefficient will be left behind after putting in 000?

It will be after differentiation nnn times, such that xnx^nxn becomes x0=1x^0 = 1x0=1.

What kind of factor will that give us?

(xn)′=nxn−1(nxn−1)′=(n−1)nxn−2((n−1)nxn−2)′=(n−2)(n−1)nxn−3… \begin{align*} (x^n)' &= nx^{n-1} \\ (nx^{n-1})' &= (n-1)nx^{n-2} \\ ((n-1)nx^{n-2})' &= (n-2)(n-1)nx^{n-3}\\ \dots & \end{align*} (xn)′(nxn−1)′((n−1)nxn−2)′…​=nxn−1=(n−1)nxn−2=(n−2)(n−1)nxn−3​

From this you might already guess the pattern. The final factor before the coefficient will be the product of all numbers from 111 up to nnn, so 1∗2∗3∗⋯∗n1 * 2 * 3 *\dots * n1∗2∗3∗⋯∗n. There is even a fancy math way to write this, called the factorial:

1∗2∗3∗⋯∗n=n! 1* 2 * 3 *\dots * n = n! 1∗2∗3∗⋯∗n=n!

To make stuff work out nicely, we define 0!=10! = 10!=1.

This works out with all previous examples.

  • a0a_0a0​ needed 000 derivatives, so the factor is 1=0!1 = 0!1=0!
  • a1a_1a1​ needed 111 derivative, so the factor is 1=1!1 = 1!1=1!
  • a2a_2a2​ needed 222 derivatives, so the factor is 2=1∗2=2!2 = 1*2 = 2!2=1∗2=2!
  • a3a_3a3​ needed 333 derivatives, so the factor is 6=1∗2∗3=3!6 = 1*2*3 = 3!6=1∗2∗3=3!

To find the coefficient, we just have to divide by the factor in front of it!

With that we can actually write down the polynomial that agrees with our given function in all derivatives at x=0x=0x=0:

p⁡(x)=f⁡(0)+f⁡′(0)x+f⁡′′(0)2x2+f⁡′′′(0)6x3+…=f⁡(0)(0)0!x0+f⁡(1)(0)1!x1+f⁡(2)(0)2!x2+f⁡(3)(0)3!x3+… \begin{align*} \operatorname{p}(x) &= \operatorname{f}(0) + \operatorname{f}'(0)x + \frac{\operatorname{f}''(0)}{2}x^2+ \frac{\operatorname{f}'''(0)}{6}x^3 + \dots \\ &= \frac{\operatorname{f}^{(0)}(0)}{0!}x^0 + \frac{\operatorname{f}^{(1)}(0)}{1!}x^1 + \frac{\operatorname{f}^{(2)}(0)}{2!}x^2 + \frac{\operatorname{f}^{(3)}(0)}{3!}x^3 + \dots \end{align*} p(x)​=f(0)+f′(0)x+2f′′(0)​x2+6f′′′(0)​x3+…=0!f(0)(0)​x0+1!f(1)(0)​x1+2!f(2)(0)​x2+3!f(3)(0)​x3+…​

Here, the notation f⁡(n)\operatorname{f}^{(n)}f(n) means the nnn-th derivative.

We can also see, that it looks very regular! The degree of the derivative corresponds to the power of xxx and the factorial!

We can write this down even more compact by using another math notation: The sum formula.

p⁡(x)=∑i=0∞f⁡(i)(0)i!xi\operatorname{p}(x) = \sum_{i=0}^{\infty}\frac{\operatorname{f}^{(i)}(0)}{i!}x^i p(x)=i=0∑∞​i!f(i)(0)​xi

This just means, that we will use a number iii as a placeholder for all integers starting from 000 (that is the i=0i=0i=0 part) and never stopping, so going to infinity (that is the top ∞\infty∞). Each of these iii will then be put into the formula to the right and the result of that is added to the whole. So if you set i=0i=0i=0 you get the first term in the formula from before. If you set i=1i=1i=1, you get the second one and so on. The ∑\sum∑ symbol is just a Greek Sigma, basically just an "S" for "sum".

While that might look scary at first, I hope you have seen what it means with the previous sections. It looks worse than it actually is!

This is more or less the formula you will find in a textbook, although there the more general form, where you don't need to start at x=0x=0x=0 is used, the idea is exactly the same though.

Of course, this is only the beginning, there is a lot more to it, but hopefully you got the general idea with this!

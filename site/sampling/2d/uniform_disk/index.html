Uniform Disc

One way to sample points in a disc is to generate a random number between 000 and rrr for the radius and another one for the angle in [0,2π][0,2\pi][0,2π]. The points won’t be uniformly distributed like that, since the area of segments on the inside is different from that of the outside. The sampling will look something like this:

function uniform_radial_circle(){

    const u_1 = Math.random();
    const u_2 = Math.random();
    
    const r = u_1;
    const alpha = u_2 * 2.0 * Math.PI;

    return [r,alpha];
}

The points are focused in the center! With uniform area sampling it looks like the following:

function uniform_circle(){

    const u_1 = Math.random();
    const u_2 = Math.random();
    
    const r = Math.sqrt(u_1);
    const alpha = u_2 * 2.0 * Math.PI;

    return [r,alpha];
}
